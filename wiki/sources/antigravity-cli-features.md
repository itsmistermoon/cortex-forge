# Antigravity CLI - Features

**URL:** https://antigravity.google/docs/cli-features  
**Original date:** 2026-06-08  
**Author:** Google Antigravity Team

## Summary

This page describes the higher-order capabilities of the CLI: work in natural language, subagent orchestration, terminal efficiency, and slash-command-based configuration. It presents Antigravity CLI as a fast terminal surface with shared agent features and a lightweight runtime model.

## Key ideas

1. The product emphasizes natural-language tasking, with subagents handling parallel work.
2. Slash commands expose plugins, MCP, skills, hooks, config, and keybindings from the terminal.
3. The CLI is positioned as a snappy, low-overhead companion to the broader Antigravity product family.
