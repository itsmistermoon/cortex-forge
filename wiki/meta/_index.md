## Structure

```
{your-vault}/
├── .raw/              ← Immutable original sources
├── wiki/
│   ├── concepts/      ← Concepts
│   ├── entities/      ← Entities
│   ├── sources/       ← Sources
│   ├── pages/         ← Projects and decisions
│   ├── meta/          ← Vault metadata
│   └── index.md       ← Master index
├── templates/         ← 5 templates
├── bin/               ← Scripts
├── skills/            ← Agent skills
├── AGENTS.md          ← Agent operating rules
└── README.md
```
