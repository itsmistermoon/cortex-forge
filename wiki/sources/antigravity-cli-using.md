# Antigravity CLI - Using AGY CLI

**URL:** https://antigravity.google/docs/cli-using  
**Original date:** 2026-06-08  
**Author:** Google Antigravity Team

## Summary

This guide covers day-to-day CLI usage: persistent settings, slash commands, keybindings, and session-navigation controls. It is the operational reference for moving between prompt-driven interaction and configuration overlays.

## Key ideas

1. Settings live in `~/.gemini/antigravity-cli/settings.json`, which is sparse and forward-compatible.
2. `/config` and `/settings` open the interactive settings panel, while launch flags can temporarily override values.
3. Slash commands and keybindings support session control, prompt editing, permissions, and navigation.
