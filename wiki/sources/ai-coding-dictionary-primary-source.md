# AI Coding Dictionary — Primary source (full article)

**URL:** https://www.aihero.dev/ai-coding-dictionary/primary-source
**Author:** Matt Pocock (AI Hero)

## Summary

Full dictionary article expanding the one-line index definition. A primary source is a source of truth in its original form — the code, the transcript, the raw log. Not an account of the thing; the thing. The article develops the cost/completeness trade-off and when to pay for it.

## Key ideas

1. An agent that reads a doc inherits the doc's staleness; an agent that reads the code is reading the current truth. When an agent asserts something wrong, ask which source class it worked from.
2. Cost is what keeps primary sources from being the default: every token billed as input and competing for attention budget.
3. What the cost buys is completeness — nothing pre-filtered by someone else's judgement. Last month's summary can't contain the detail that turned out to matter today; the primary source still does.
4. Reach for the primary source when precision matters (the exact signature, the actual error). Much of managing context is deciding when to pay for it.
