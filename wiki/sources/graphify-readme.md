# graphify — README (v8)

**URL:** https://github.com/safishamsi/graphify
**Original date:** 2026-06-12
**Author:** Safi Shamsi / Graphify Labs (YC S26)

## Summary

Official README of graphify v8: AI coding assistant skill that turns any folder into a queryable knowledge graph. 66.3k stars, YC S26, 733 commits, 134 releases, MIT license. Supports 20+ platforms with per-platform install commands. CommandCode is NOT in the supported list.

## Key ideas

1. Multi-agent install with per-platform commands (`graphify install --platform <name>`)
2. Hook mechanisms vary: PreToolUse (Claude Code, Codex, CodeBuddy), BeforeTool (Gemini CLI), native plugin (Kilo Code), alwaysApply (Cursor), AGENTS.md fallback (Trae)
3. CommandCode gap: not in the supported list — cortex-forge fills this
