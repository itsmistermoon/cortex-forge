# Antigravity CLI - Tutorial

**URL:** https://antigravity.google/docs/cli-tutorial  
**Original date:** 2026-06-08  
**Author:** Google Antigravity Team

## Summary

This tutorial shows the first-run flow for Antigravity CLI: launch `agy`, authenticate, give the agent a task, review the artifact it produces, and verify the result from the terminal. It frames the CLI as a guided loop of instruction, review, execution, and confirmation.

## Key ideas

1. First launch centers on silent authentication, with browser-based sign-in as the fallback.
2. The CLI is designed to produce artifacts the user can inspect before accepting changes.
3. Verification happens in the terminal, keeping the agent loop inside the shell workflow.
